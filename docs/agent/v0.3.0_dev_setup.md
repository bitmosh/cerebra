# v0.3.0 Dev Setup — fossic-py

## What's new

Phase 6 Step 1 adds fossic-py as a production dependency (fossic v1.0-rc.1, Rust-core event sourcing).

## Installing fossic-py

fossic-py is a PyO3-built Rust extension. It ships as a pre-built `.so` at:

```
/home/boop/Projects/fossic/fossic-py/python/fossic/_fossic.cpython-312-x86_64-linux-gnu.so
```

Install into the Cerebra venv using editable mode:

```bash
uv pip install -e /home/boop/Projects/fossic/fossic-py
```

This installs `fossic==0.1.0` into the venv. Verify:

```bash
uv run python -c "import fossic; print(fossic.__file__)"
```

## pyproject.toml

The dependency is declared as a path dependency:

```toml
"fossic @ file:///home/boop/Projects/fossic/fossic-py",
```

mypy is configured to ignore fossic imports (Rust extension, no stubs):

```toml
[[tool.mypy.overrides]]
module = ["fossic", "fossic.*"]
ignore_missing_imports = true
```

## When the .so is not built

If you need to rebuild the fossic-py extension (e.g., on a new machine or after Rust source changes):

```bash
cd /home/boop/Projects/fossic/fossic-py
maturin develop --release
```

This requires `maturin` and the Rust toolchain. After rebuilding, re-run `uv pip install -e .` in the cerebra venv.

## Note on system Python

`python` in this repo uses the system Python which does NOT have fossic installed. Always use `uv run python` to access the cerebra venv where fossic is available.

## Ollama (for integration tests)

Phase 6 Step 2 integration tests hit local Ollama directly. To run them:

1. Start Ollama: `ollama serve` (or via the AI stack: `cd ~/Projects/ai-stack && docker compose up -d`)
2. Pull the model: `ollama pull huggingface.co/unsloth/granite-4.1-3b-GGUF:Q4_K_M`
3. Run integration tests: `uv run pytest -m integration`

Skip Ollama integration tests without CEREBRA_SKIP_OLLAMA:
```bash
uv run pytest -m "not integration"
```

Or force skip even when Ollama is running:
```bash
CEREBRA_SKIP_OLLAMA=1 uv run pytest
```

The first call after cold start may take up to 164s (model load). Subsequent calls are 1–3s.

## Pyright

Pyright reports `reportMissingImports` for fossic since it can't resolve the Rust extension. This is expected — mypy (which CI runs) is configured to suppress this. The Pyright warnings in the IDE are cosmetic.
